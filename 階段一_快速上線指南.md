## 🌐 第二步：部署到線上（10 分鐘）

### 推薦方案：GitHub Pages（完全免費）

#### 2.1 創建 GitHub 帳號（如果沒有）
- 訪問 https://github.com
- 註冊免費帳號

#### 2.2 上傳項目

**方式 A：使用 GitHub Desktop（推薦，無需命令行）**

1. 下載並安裝 [GitHub Desktop](https://desktop.github.com/)

2. 打開 GitHub Desktop，登入帳號

3. 創建新倉庫：
   - File → New Repository
   - Name: `orbit-app`
   - Local Path: 選擇您的 `Orbit_Final` 資料夾的上層目錄
   - 勾選 "Initialize this repository with a README"
   - 點擊 "Create Repository"

4. 發佈到 GitHub：
   - 點擊 "Publish repository"
   - 取消勾選 "Keep this code private"（保持公開）
   - 點擊 "Publish repository"

**方式 B：使用網頁上傳**

1. 訪問 https://github.com/new

2. 創建新倉庫：
   - Repository name: `orbit-app`
   - Public（公開）
   - 點擊 "Create repository"

3. 上傳文件：
   - 點擊 "uploading an existing file"
   - 將所有文件拖放到頁面
   - 點擊 "Commit changes"

#### 2.3 啟用 GitHub Pages

1. 進入倉庫設置：
   - 點擊 "Settings"（齒輪圖標）

2. 找到 Pages 設置：
   - 左側選單找到 "Pages"

3. 配置發布源：
   - Source: 選擇 "Deploy from a branch"
   - Branch: 選擇 "main"
   - Folder: 選擇 "/ (root)"
   - 點擊 "Save"

4. 等待部署（約 1-2 分鐘）
   - 頁面會顯示您的網址
   - 格式：`https://你的用戶名.github.io/orbit-app/`

5. **重要**：更新 manifest.json
   ```json
   {
     "start_url": "https://你的用戶名.github.io/orbit-app/",
     "scope": "https://你的用戶名.github.io/orbit-app/"
   }
   ```
   然後重新上傳

## 📱 第三步：生成分享 QR Code（2 分鐘）

### 線上生成（推薦）

1. **訪問 QR Code 生成器**
   - https://www.qr-code-generator.com/
   - 或 https://qrcode.tec-it.com/zh

2. **輸入您的網址**
   ```
   https://你的用戶名.github.io/orbit-app/
   ```

3. **自定義 QR Code**
   - 顏色：選擇 #8B4513（木質調）
   - Logo：可以上傳您的應用圖標
   - 格式：PNG 或 SVG

4. **下載並分享**
   - 下載高清 QR Code
   - 分享給朋友測試

## 🆘 常見問題

### Q: 部署後圖標還是默認圖標？
A: 
1. 確認 icons\ 資料夾已上傳
2. 檢查 manifest.json 路徑正確
3. 清除瀏覽器緩存
4. 重新安裝應用

### Q: 朋友說無法安裝？
A: 
- Android: 確認使用 Chrome
- iOS: 必須使用 Safari
- 檢查是否有 HTTPS（GitHub Pages 自動支援）

### Q: 更新後朋友看不到新版本？
A: 
1. 更新 service-worker.js 中的版本號
2. 讓用戶關閉應用重開
3. 或清除應用數據後重新安裝

### Q: 數據會丟失嗎？
A: 
- 數據保存在瀏覽器 localStorage
- 只要不清除瀏覽器數據就不會丟失
- 建議提醒用戶定期備份
