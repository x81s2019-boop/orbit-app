✅ 步驟一：生成應用圖標（5 分鐘）
─────────────────────────────────────────
□ 打開 generate-icons.html
□ 等待圖標自動生成
□ 點擊「下載所有圖標」
□ 將 8 個 PNG 文件放入 icons\ 資料夾
□ 確認文件名正確：
  □ icon-72x72.png
  □ icon-96x96.png
  □ icon-128x128.png
  □ icon-144x144.png
  □ icon-152x152.png
  □ icon-192x192.png
  □ icon-384x384.png
  □ icon-512x512.png

═══════════════════════════════════════════

✅ 步驟二：註冊 GitHub（5 分鐘）
─────────────────────────────────────────
□ 訪問 https://github.com
□ 點擊 Sign up
□ 填寫資料（Email、密碼、用戶名）
□ 驗證 Email
□ 完成註冊

如果已有帳號，跳過此步驟 →

═══════════════════════════════════════════

✅ 步驟三：上傳項目（10 分鐘）
─────────────────────────────────────────

方式 A：使用 GitHub Desktop（推薦）
□ 下載 GitHub Desktop: https://desktop.github.com/
□ 安裝並登入
□ File → New Repository
  □ Name: orbit-app
  □ Local Path: 選擇 Orbit_Final 的上層目錄
  □ Create Repository
□ Publish repository
  □ 取消勾選 "Keep this code private"
  □ Publish

方式 B：網頁上傳
□ 訪問 https://github.com/new
□ Repository name: orbit-app
□ 選擇 Public
□ Create repository
□ 點擊 "uploading an existing file"
□ 拖放所有文件
□ Commit changes

═══════════════════════════════════════════

✅ 步驟四：啟用 GitHub Pages（2 分鐘）
─────────────────────────────────────────
□ 進入倉庫 Settings
□ 左側選單找到 Pages
□ Source: Deploy from a branch
□ Branch: main
□ Folder: / (root)
□ Save
□ 等待 1-2 分鐘
□ 複製網址（格式：https://用戶名.github.io/orbit-app/）

□ 更新 manifest.json：
  □ 打開 manifest.json
  □ 修改 start_url 和 scope 為您的網址
  □ 重新上傳到 GitHub

═══════════════════════════════════════════

✅ 步驟五：生成 QR Code（3 分鐘）
─────────────────────────────────────────
□ 訪問 https://www.qr-code-generator.com/
□ 輸入您的網址
□ 自定義顏色（可選）：#8B4513
□ 下載 QR Code（PNG 格式）
□ 保存到電腦

═══════════════════════════════════════════

✅ 步驟六：測試應用（5 分鐘）
─────────────────────────────────────────

桌面測試：
□ 在 Chrome/Edge 打開您的網址
□ 檢查所有功能正常
□ 點擊地址欄的安裝圖標 ➕
□ 安裝為應用
□ 測試離線功能

手機測試（Android）：
□ 掃描 QR Code
□ 點擊「添加到主屏幕」
□ 確認圖標正確
□ 測試所有功能

手機測試（iOS）：
□ 用 Safari 掃描 QR Code
□ 點擊分享 → 加入主畫面
□ 確認圖標正確
□ 測試所有功能
